# Privacy & Mobile Tools / Приватность и мобильные инструменты

This repository contains resources for ad blocking, privacy configuration, and Android system debloating.

Этот репозиторий содержит ресурсы для блокировки рекламы, настройки конфиденциальности и очистки системы Android.

## Ad Blocking & Security / Блокировка рекламы и безопасность

Software and guides for enhancing online privacy and blocking unwanted content.

Программное обеспечение и руководства для повышения конфиденциальности в Интернете и блокировки нежелательного контента.

* **AdGuard:** https://adguard.com/ru/welcome.html

    * Продвинутое программное обеспечение для блокировки рекламы и защиты от отслеживания.
* **StellarSand Privacy Settings:** https://github.com/StellarSand/privacy-settings?tab=readme-ov-file#privacy-settings
    * Comprehensive guide and scripts for hardening system privacy settings.
    * Подробное руководство и скрипты для усиления настроек конфиденциальности системы.

## Android Optimization / Оптимизация Android

Tools for removing bloatware and optimizing Android devices.

Инструменты для удаления предустановленного ПО и оптимизации устройств Android.

* **Universal Android Debloater (Next Gen):** https://github.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation
    * A GUI tool to remove unnecessary pre-installed apps (bloatware) from Android devices via ADB without root.
    * Графический инструмент для удаления ненужных предустановленных приложений (bloatware) с устройств Android через ADB без root-прав.
